# dayjs-extends

[![NPM version](https://img.shields.io/npm/v/dayjs-extends)](https://www.npmjs.com/package/dayjs-extends)

dayjs extends

## Install

```bash
npm i dayjs-extends
```
## Usage


```js
import dayjsExt from 'dayjs-extends'

const de = dayjsExt()
console.log(de.sayHi()) // hello, dayjs extends!
```
